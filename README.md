<h2>Idea</h2>
The website is designed to be a simple recipe holder with fast, intuitive front-end. 

<h2>Purpose</h2>
This is a practice project to get hang of Django, Python, CSS, HTML, JavaScript, jQuery, Ajax, etc. properly.
